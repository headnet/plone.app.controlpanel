Mail control panel
==================

First some initial setup code:

    >>> from Products.CMFCore.utils import getToolByName
    >>> from zope.component import getUtility
    >>> from Products.MailHost.interfaces import IMailHost
    >>> mailhost = getUtility(IMailHost)

    >>> from plone.testing.z2 import Browser
    >>> browser = Browser(layer['app'])

Login as manager
    >>> loginAsManager(browser)

Viewing the mail control panel
--------------------------------

    >>> browser.open('http://nohost/plone/@@mail-controlpanel')
    >>> browser.url.endswith('mail-controlpanel')
    True

Click the cancel button:

    >>> browser.getControl(name="form.actions.cancel").click()
    >>> browser.url.endswith('plone_control_panel')
    True

There should be still no changes:

    >>> 'Changes canceled.' in browser.contents
    True

Make some changes
-----------------

    >>> browser.open('http://nohost/plone/@@mail-controlpanel')
    >>> browser.url.endswith('mail-controlpanel')
    True

    >>> browser.getControl(name='form.smtp_host').value = 'localhost2'
    >>> browser.getControl(name='form.smtp_port').value = '2525'
    >>> browser.getControl(name='form.smtp_userid').value = 'admin'
    >>> browser.getControl(name='form.smtp_pass').value = 'secret'
    >>> browser.getControl(name='form.email_from_name').value = 'Spambot'
    >>> browser.getControl(name='form.email_from_address').value = 'spam@localhost'

Click the save button:

    >>> browser.getControl(name="form.actions.save").click()
    >>> browser.url.endswith('mail-controlpanel')
    True

We should be informed that something has changed:

    >>> 'Changes saved.' in browser.contents
    True

Make sure the changes have been applied correctly to the mailhost:

    >>> mailhost.smtp_host
    u'localhost2'

    >>> mailhost.smtp_port
    2525

    >>> getattr(mailhost, 'smtp_userid', mailhost.smtp_uid)
    u'admin'

    >>> getattr(mailhost, 'smtp_pass', mailhost.smtp_pwd)
    u'secret'

    >>> ptool = getToolByName(layer['portal'], 'portal_properties')
    >>> site_props = ptool.site_properties
    >>> site_props.email_from_name
    u'Spambot'

    >>> site_props.email_from_address
    'spam@localhost'

Dealing with non-ascii
------------------------

There is no problem if we use the control panel to fill in the values:

    >>> browser.open('http://nohost/plone/@@mail-controlpanel')

    >>> browser.getControl(name='form.email_from_name').value = '\xc3\x89\xc3\xa7 \xc3\xb6\xc5\x93 \xc3\xb2'

    >>> browser.getControl(name="form.actions.save").click()
    >>> browser.url.endswith('mail-controlpanel')
    True

    >>> 'Changes saved.' in browser.contents
    True

    >>> site_props.email_from_name
    u'\xc9\xe7 \xf6\u0153 \xf2'

But we must deal with encoding if we someone set this info via ZMI:

    >>> browser.open('http://nohost/plone/manage_propertiesForm')
    >>> browser.getControl(name='email_from_name:string').value = '\xc3\x89\xc3\xa7 \xc3\xb6\xc5\x93 \xc3\xb2'
    >>> browser.getControl(name="manage_editProperties:method").click()

And the control panel should not break:
    
    >>> browser.open('http://nohost/plone/@@mail-controlpanel')

Even though data is stored as string:

    >>> site_props.email_from_name
    '\xc3\x89\xc3\xa7 \xc3\xb6\xc5\x93 \xc3\xb2'
